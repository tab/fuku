# fuku

**fuku** is a lightweight CLI orchestrator for running and managing multiple local services in development environment.
It’s designed for speed, simplicity, and readability

## Features

- Read configuration from `fuku.yaml`
- Start multiple services with a single command
- Minimal configuration with sensible defaults
- Clear, structured logging

## About name

The name fuku (福) means `good fortune` or `blessing` in Japanese.
It’s also inspired by the legendary Japanese jazz pianist Ryo Fukui.
